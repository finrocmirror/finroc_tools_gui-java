
Finroc GUI

Usage: fingui [options] [guifile]
Start fingui using specified options and optionally loading specified file. 

available options:

--classic   Use Java Swing classic look and feel
--connect=[<connection type>:]<address>

   Connect to specified address using specified connection type. If no
   connection type is specified, "TCP ports only" is used.
   Example: fingui --connect=TCP:localhost:4444
